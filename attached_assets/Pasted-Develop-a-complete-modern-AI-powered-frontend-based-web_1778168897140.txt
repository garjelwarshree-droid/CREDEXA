Develop a complete modern AI-powered frontend-based web application titled “Smart Card Comparison & Recommendation Website Using Association Rules”.

The application should help users discover, compare, filter, and get smart recommendations for different types of financial cards using Association Rule Mining without any backend. The entire project should work using frontend technologies, static datasets, and JavaScript-based recommendation logic.

The platform should support:
- Credit Cards
- Debit Cards
- Cashback Cards
- Travel Cards
- Fuel Cards
- Shopping Cards
- Student Cards
- Premium Cards
- Virtual Cards

Main Goal:
Build a smart recommendation system using Association Rules where users receive card suggestions based on predefined patterns and preferences.

Use:
- Apriori Algorithm
- Frequent Itemset Mining
- Rule-Based Recommendation Logic
- Support
- Confidence
- Lift

Example Rules:
- Users interested in Fuel Cards also prefer Cashback Cards
